# 配置 OpenAI 密钥

> 如果你不知道 OpenAI 是什么，可以参考 [OpenAI 官网](https://www.openai.com/)。
>
> 简而言之，OpenAI 提供了强大且实惠的人工智能 API，可以用来做各种有趣的事情。

OpenAI 是 DashPlayer 的核心 AI 服务，配置后即可使用以下全部功能：

- **字幕翻译** — 使用大语言模型将英文字幕翻译为中文（或其他语言）
- **语音转文字** — 使用 `whisper-1` 模型从视频中自动生成英文字幕
- **整句学习** — 对字幕句子进行语法分析、短语解释等深度学习
- **单词词典** — 鼠标悬停在单词上，通过 AI 获取详细释义
- **单词发音** — 使用 OpenAI TTS（`tts-1`）朗读单词发音

## 获取 API 密钥

OpenAI 官方接口在部分地区可能不便获取，您可以通过第三方中转服务来获取 API 密钥，通常价格更实惠：

- https://uniapi.ai/
- https://aihubmix.com/

注册充值后即可获得 API Key 和 Endpoint。

## 在 DashPlayer 中配置

1. 打开 DashPlayer 设置页面，进入「服务凭据」
2. 在 OpenAI 区域填写 **API Key** 和 **Endpoint**
3. 点击「测试连接」按钮验证配置是否正确

## 配置可用模型

在服务凭据页面的 OpenAI 区域下方，您可以管理可用的模型列表：

1. 在输入框中填写模型名称（如 `gpt-4o-mini`），点击「添加」
2. 添加的模型会显示在模型表格中

<!-- 截图占位：模型管理表格 -->

## 为功能分配模型

进入「功能设置」页面，可以为每项功能分别选择使用的模型：

- **字幕翻译** — 选择用于翻译字幕的模型
- **单词词典** — 选择用于单词释义的模型
- **整句学习** — 选择用于句子分析的模型

不同的模型在效果和价格上有所差异，您可以根据需求灵活选择。

<!-- 截图占位：功能设置页面 -->

> 请确保您使用的 API 服务支持所配置的模型，以及 `whisper-1`（语音转文字）和 `tts-1`（单词发音）接口。
