# 词汇学习工坊

词汇学习工坊就是把当前视频里的生词句子自动裁成小片段，方便后面集中回看。

## 具体怎么做

### 1. 先打开视频和字幕

先像平时一样打开视频，并加载对应字幕。这个功能依赖字幕，没有字幕就不能裁切。

### 2. 点击“裁切生词视频”

在播放器页面点击<control>裁切生词视频</control>。

DashPlayer 会自动：

- 扫描当前视频的字幕
- 找出包含生词的句子
- 把这些句子裁成小片段
- 保存到词汇学习工坊

如果当前视频里没有可裁切的片段，按钮会直接提示你。

![cut-word-clip.png](cut-word-clip.png)

### 3. 等它裁切完成

裁切完成后，这些片段会自动出现在 <ui-path>Vocabulary Studio</ui-path>。

### 4. 去词汇学习工坊回看

打开 <ui-path>Vocabulary Studio</ui-path> 页面，就能看到刚刚生成的片段。

你可以这样用：

- 点击左边单词，筛选这个单词相关的片段
- 点击右边的视频卡片，开始播放片段
- 点击字幕中的某一句，跳回那一句重新听

![word-studio.png](word-studio.png)

## 如果没有看到片段

正常使用时不需要手动恢复。

只有在异常情况下，比如你已经裁切过片段，但词汇学习工坊页面还是空的，才需要点击：

- 词汇学习工坊页面里的“从本地恢复词汇工坊片段”
- 或者进入 <ui-path>Settings Center / 存储设置</ui-path>，点击“重新同步词汇工坊”

这样会把之前已经创建过的词汇工坊片段记录，从本地文件重新恢复出来。
