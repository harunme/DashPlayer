# 配置 API

DashPlayer 的 AI 功能需要配置 API 密钥才能使用。

## 功能概览

DashPlayer 提供以下 AI 驱动的功能：

| 功能 | 说明 | 可选服务 |
|------|------|----------|
| **字幕翻译** | 将英文字幕翻译为中文或其他语言 | OpenAI / 腾讯云 |
| **语音转文字** | 从视频中自动生成字幕 | OpenAI（whisper-1）/ 本地 Whisper |
| **整句学习** | 对字幕句子进行语法分析、短语解释等 | OpenAI |
| **单词词典** | 鼠标悬停在单词上查看释义 | OpenAI / 有道 |
| **单词发音** | 单词的 TTS 语音朗读 | OpenAI |

## 推荐配置

**只需配置 OpenAI 即可使用全部功能。** OpenAI 覆盖了字幕翻译、语音转文字、整句学习、单词词典和单词发音五项功能，是最推荐的配置方式。

有道和腾讯云为可选配置：
- **有道云** — 可作为单词词典的替代方案
- **腾讯云** — 可作为字幕翻译的替代方案

## 配置步骤

1. 在 DashPlayer 设置页面中，进入「服务凭据」页面，填写对应服务的 API 密钥
2. 进入「功能设置」页面，为每项功能选择使用的服务引擎

详细的 API 申请步骤请参考各服务的配置文档：

- [配置 OpenAI 密钥](Config-OpenAI-API.md)（推荐）
- [配置有道云密钥](Config-YouDao-API.md)（可选）
- [配置腾讯云密钥](Config-Tencent-API.md)（可选）
